## Question

Create a Case validation rule that requires the **Customer Reference ID** field to be numeric when entered.

- Object: **Case**
- Field: Customer Reference ID (`Customer_Reference_ID__c`)
- Rule Name: `CASE002_Customer_Ref_Numeric`
- Description: Ensure Customer Reference ID contains only numbers when users populate it.
- Requirement: If `Customer_Reference_ID__c` is **not blank**, it must be **numeric**.
- Hint: Use `ISBLANK`, `NOT`, `ISNUMBER`.
- Error Message: `Customer Reference ID must contain only numbers [CASE002]`
- Error Location: `Customer Reference ID`

---

## Solution

<details>
    <summary><strong>Click here!</strong></summary>

    ```
    AND(
    NOT(ISBLANK(Customer_Reference_ID__c)),
    NOT(ISNUMBER(Customer_Reference_ID__c))
    )
    ```

</details>
